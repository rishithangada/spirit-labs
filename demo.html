External attack surface mapping

The port scanner sweeps 20 common service ports and flags anything that expands your attack surface — unencrypted services, exposed databases, and remote access endpoints that should not be public.

  • 20 ports checked: FTP, SSH, Telnet, HTTP/S, RDP, MySQL, Redis, and more
  • Risky service detection with severity rating
  • Remediation step included for every open finding
  • Output maps directly to OWASP A05: Security Misconfiguration

HTTP security header gap detection

The header inspector checks every security-relevant HTTP response header and flags gaps. Missing or misconfigured headers are one of the easiest wins in web security — and one of the most commonly overlooked.

  • CSP, HSTS, X-Frame-Options, Referrer-Policy, Permissions-Policy
  • Each finding includes the recommended header value
  • Score out of 6 for quick executive communication
  • Maps to OWASP A05: Security Misconfiguration

TLS/SSL certificate and cipher review

The SSL checker validates your certificate chain, checks expiry, and reviews the supported protocol versions and cipher suites. Outdated protocols like TLS 1.0 are still common and still exploitable.

  • Certificate validity and expiry countdown
  • Supported protocol check: TLS 1.0, 1.1, 1.2, 1.3
  • Cipher suite review with specific remediation
  • Letter grade for quick executive communication
